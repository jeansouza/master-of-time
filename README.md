<h3>How to install?</h3>

You can download extension from link above.

If you want install it manually, clone project to ~/.local/share/gnome-shell/extensions/master-of-time-extension@rostegg.github.com folder and restart gnome desktop (Alt + F2 -> r -> Enter)

<h3>Bug report</h3>

For bug report provide gnome shell version (gnome-shell --version) and error message, if installation failed (Alt+F2 -> lg -> Extensions -> Show Errors).
